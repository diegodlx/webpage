# Diego Loyola | Portfólio Profissional

Portfólio pessoal desenvolvido para apresentar minha trajetória, competências técnicas e projetos em desenvolvimento web, com foco em Backend e construção de aplicações modernas.

Este projeto foi criado com uma proposta objetiva: servir como vitrine profissional no GitHub e como página de apresentação para recrutadores, parceiros e potenciais oportunidades.

## Visão Geral

O site reúne, em uma interface simples, responsiva e direta:

- apresentação profissional;
- resumo da minha transição e experiência;
- destaques de projetos desenvolvidos;
- links para GitHub, LinkedIn, WhatsApp e e-mail;
- opção de download do currículo.

## Preview do Projeto

O portfólio foi estruturado em seções bem definidas:

- `Início`: apresentação principal com foto, nome, especialidade e acesso ao currículo;
- `Sobre Mim`: resumo da experiência profissional e direcionamento de carreira;
- `Portfólio`: exibição de projetos com descrição e link para repositórios;
- `Contato`: canais diretos para conexão profissional.

## Tecnologias Utilizadas

Este projeto foi desenvolvido com foco em front-end estático, sem dependências externas, utilizando:

- `HTML5`
- `CSS3`
- `CSS Variables`
- `Flexbox`
- `Media Queries`
- `Animações CSS`

## Principais Recursos Implementados

- layout responsivo para diferentes tamanhos de tela;
- menu de navegação adaptado para dispositivos móveis;
- cabeçalho fixo para facilitar a navegação;
- rolagem suave entre seções com `scroll-behavior: smooth`;
- cards de projetos com destaque visual;
- uso de variáveis CSS para padronização de cores;
- efeitos de hover em botões, links e ícones;
- animação visual aplicada ao logo com base em rolagem;
- estrutura simples e de fácil manutenção.

## Projetos Destacados no Portfólio

Atualmente, a página apresenta os seguintes projetos:

### 1. Cadastro de Usuários

Aplicação Full Stack de cadastro de usuários com integração a banco de dados PostgreSQL, utilizando Node.js, TypeScript e Express.

### 2. API da Champions League

API REST desenvolvida com Node.js, Express e TypeScript para gerenciamento de dados relacionados à Champions League.

### 3. Calculadora IA

Aplicação moderna desenvolvida com React, TypeScript e Styled Components, com apoio de Inteligência Artificial no processo de construção.

## Responsividade

O projeto foi desenvolvido com abordagem responsiva para oferecer boa experiência de navegação em:

- smartphones;
- tablets;
- notebooks;
- monitores maiores.

Os ajustes são realizados com `media queries`, adaptando navegação, tamanhos de fonte, imagens, botões e organização dos blocos de conteúdo.

## Como Executar o Projeto

Por se tratar de um projeto estático, basta abrir o arquivo `index.html` em qualquer navegador.

### Opção 1

Acessar diretamente do link do GitHub Pages:

[diegodlx.github.io/webpage](https://diegodlx.github.io/webpage/)

### Opção 2

Abrir diretamente o arquivo:

```bash
index.html
```

### Opção 3

Se preferir, utilizar uma extensão como Live Server no VS Code para rodar localmente.
